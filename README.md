# Hospital Emergency Room Dashboard

## Project Overview
This project is an interactive Hospital Emergency Room Dashboard created to analyze patient records, admission status, waiting time, satisfaction score, gender distribution, and department referrals.

The dashboard helps in monitoring hospital performance and understanding patient trends using visual analytics.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Dashboard Design
- Data Cleaning

## Dashboard Features
- Monthly Patient Report
- Patient Admission Analysis
- Average Waiting Time Tracking
- Patient Satisfaction Score
- Gender-wise Analysis
- Department Referral Analysis
- Age Group Patient Distribution
- Year Filter (2023 & 2024)

## Key Insights
- Total patients analyzed: 494
- 47% patients were admitted
- Average waiting time was 36.39 minutes
- Most patients came without department referral
- Age group 10–39 showed higher patient count

## Conclusion
This dashboard provides meaningful healthcare insights through data visualization and helps improve operational decision-making in emergency room managemen
